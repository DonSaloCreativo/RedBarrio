# 🦁 LlamaBarrio - Datos Reales de la Comunidad

Plataforma web para compartir datos útiles, cercanos y compartidos por vecinos de tu comuna.

## 🎯 Características

- 🗺️ Búsqueda de locales por comuna
- 🍽️ Recomendaciones de joyitas (datos reales)
- 📱 Diseño responsive (móvil, tablet, desktop)
- 🔗 Integración con Tally para formularios

## 🚀 Cómo usar

1. Abre `index.html` en tu navegador
2. O accede a la versión online (si tienes hospedaje)
